# Node.js Express

Express.js là framework web phổ biến nhất cho Node.js, giúp xây dựng APIs nhanh chóng.

## Tính năng chính

- **Routing**: Xử lý HTTP requests
- **Middleware**: Chức năng mở rộng
- **Template engines**: Render views
- **Error handling**: Xử lý lỗi robust
