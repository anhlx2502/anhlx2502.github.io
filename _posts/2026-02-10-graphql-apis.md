# GraphQL APIs

GraphQL là query language cho APIs, cho phép clients request exactly data cần.

## Advantages over REST

- **Single endpoint**: Một endpoint thay nhiều
- **Precise data**: Client specify fields
- **Versioning**: Không cần version APIs
- **Introspection**: Self-documenting APIs
