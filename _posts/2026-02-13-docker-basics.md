# Docker Cơ Bản

Docker là nền tảng containerization phổ biến nhất, cho phép đóng gói ứng dụng và dependencies vào một container nhẹ, di động.

## Lợi ích chính

- **Isolation**: Mỗi container chạy độc lập
- **Portability**: Chạy mọi nơi có Docker
- **Efficiency**: Tiết kiệm tài nguyên hơn VM
