# React Hooks

React Hooks là tính năng mới từ React 16.8, cho phép sử dụng state và lifecycle trong functional components.

## Hooks phổ biến

- **useState**: Quản lý state local
- **useEffect**: Side effects và lifecycle
- **useContext**: Truy cập context
- **useReducer**: State phức tạp
