# Git Workflow

Git là hệ thống version control phân tán, thiết yếu cho development hiện đại.

## Best Practices

- **Branching strategy**: Feature branches, main/master
- **Commit messages**: Descriptive và conventional
- **Pull requests**: Code review trước merge
- **Rebasing**: Clean history
