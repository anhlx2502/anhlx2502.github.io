# TypeScript

TypeScript là superset của JavaScript với static typing, giúp code robust hơn.

## Tính năng chính

- **Static typing**: Catch errors lúc compile
- **Interfaces**: Define object shapes
- **Generics**: Reusable code
- **Decorators**: Meta programming
