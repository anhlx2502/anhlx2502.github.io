# AWS Cloud Services

AWS cung cấp hàng trăm services cho mọi nhu cầu cloud computing.

## Core Services

- **EC2**: Virtual servers
- **S3**: Object storage
- **Lambda**: Serverless functions
- **RDS**: Managed databases
